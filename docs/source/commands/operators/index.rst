Operator commands
====================

These commands *operate* on device to acheive a certain task &
represent the basic blitz actions and are primarily used to autogenerate
these blitz actions snippets.

* :ref:`execute` - Execute commands; autogenerates *'execute'* blitz action
* :ref:`configure` - Configure CLI; autogenerates *'configure'* blitz action
* :ref:`parse` - Parse commands; autogenerates *'parse'* blitz action
* :ref:`api` - Invoke Genie API's; autogenerates *'api'* blitz action
* :ref:`run_genie_sdk` - Run generated blitz testcases; autogenerates *'run_genie_sdk'* blitz action
* :ref:`sleep` - Sleep; autogenerates *'sleep'* blitz action
