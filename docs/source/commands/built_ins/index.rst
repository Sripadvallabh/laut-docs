Builtin commands
====================

These commands come with a packaged cmd2 application.

   * :doc:`alias` - Manage aliases for commands
   * :doc:`edit` - Run a text editor and optionally open a file with it
   * :doc:`history` - View, run, edit, save, or clear previously entered commands
   * :doc:`macro` - Manage macros for commands
   * :doc:`py` - Launch an interactive Python shell
   * :doc:`run_script` - Run a script file consisting of LAUT shell commands
   * :doc:`run_pyscript` - Run a python script file inside LAUT shell
   * :doc:`set` - Modify & view LAUT settings
   * :doc:`shell` - Execute a command as if at the OS prompt
   * :doc:`shortcuts` - List available shortcuts

Read `cmd2 builtin commands <https://cmd2.readthedocs.io/en/latest/features/builtin_commands.html>`_ to learn more.
