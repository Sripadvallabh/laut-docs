Management commands
====================

These commands handle the management of a testbed & its devices.

They are primarily used to load testbeds, connect & disconnect devices as
well as switch between multiple devices in a testbed.

* :ref:`testbed` - Load testbed onto LAUT shell
* :ref:`device` - Load device onto LAUT shell
* :ref:`exit` - Exit LAUT shell
